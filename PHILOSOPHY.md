# Philosophy

The principles that guide BumbaClaude.

---

## Freedom is the foundation

BumbaClaude exists because developers deserve to orchestrate AI agents without hacks, workarounds, or tools that might get pulled from under them. Real freedom comes from building on solid ground.

## The wrapper principle

We never modify what we wrap. Three interfaces, all official: tmux sessions, filesystem reads, Agent SDK. If it requires patching Claude Code, it's out of scope. This isn't a limitation; it's the architecture.

## Build on solid ground

Tools that exploit loopholes break when the loophole closes. BumbaClaude uses the same interfaces Claude Code exposes to every user. No special access. No hidden tricks.

## Your access, your choice

Subscription or API, your call. BumbaClaude doesn't care how you access Claude Code. It orchestrates whatever you've got.

## Open source, open road

MIT licensed. Fork it. Modify it. Build on it. The code is yours.
